# Probability Simulator

This project simulates repeated dice rolls to analyze probability distributions and expected value using a Monte Carlo approach.

## Overview
The program allows the user to input the number of trials and simulates rolling a fair six-sided die. It calculates:
- Experimental average
- Theoretical expected value
- Difference between observed and expected values
- Frequency distribution of outcomes

## Key Concepts Demonstrated
- Expected Value
- Law of Large Numbers
- Probability Distributions
- Simulation using random variables

## Example
With a small number of trials, results vary due to randomness. As trials increase, the experimental average approaches the theoretical expected value of 3.5.

## Technologies Used
- Java
- Arrays and loops
- Scanner for user input

## Why I Built This
I built this project to better understand how theoretical probability compares to simulated outcomes and how randomness behaves over repeated trials.
